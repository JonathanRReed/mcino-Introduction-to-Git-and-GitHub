# Contributing

All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

Thank you for helping improve the Simple Interest Calculator project.

## How to Contribute

1. Fork the repository.
2. Create a branch for your change.
3. Make focused updates with clear commit messages.
4. Test your changes before opening a pull request.
5. Open a pull request with a concise summary of the change.

## Reporting Issues

When reporting a bug, include the steps to reproduce the issue, the expected result, and the actual result.

## Pull Request Guidelines

- Keep changes small and focused.
- Update documentation when behavior changes.
- Follow the existing shell script style.
- Be respectful in all project discussions.
